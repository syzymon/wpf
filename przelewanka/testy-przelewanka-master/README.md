# Testy do zadania Przelewanka

- Testy należy dodawać do katalogu tests. W postaci kodu źródłowego ocaml wykorzystującego bibliotekę `Przelewanka` i wypisującego na `stderr` informacje o wykrytych problemach.
- Aby użyć skryptu `testuj.sh` do weryfikacji tetsów należy umieścić swój program w katalogu ze skryptem w pliku `przelewanka.ml` i uruchomić skrypt
